# Car Marketplace (Standalone Repo Draft)

هذا المجلد مقصود كخطوة انتقالية فقط حتى يتم نقل مشروع سوق السيارات إلى **مستودع مستقل**.

## لماذا؟
- لتجنب خلط مشروع المتجر الحالي مع مشروع سوق السيارات.
- تسهيل التطوير والنشر والصيانة لكل مشروع بشكل منفصل.

## الخطوة القادمة المقترحة
1. إنشاء مستودع جديد باسم مثل: `car-marketplace`.
2. نقل صفحات سوق السيارات وملفاتها إلى المستودع الجديد فقط.
3. إبقاء هذا المستودع مخصصًا للمتجر الحالي بدون واجهات سوق السيارات.

## ملاحظة
تم في هذا التعديل **استرجاع الواجهة الأصلية** داخل المشروع الحالي حسب الطلب.
